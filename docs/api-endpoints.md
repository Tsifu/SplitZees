# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Friends

- `GET /api/friends`
- `POST /api/friends`


### Bills

- `POST /api/bills`
- `PATCH /api/bills/${id}`
- `DELETE /api/bills/${id}`
